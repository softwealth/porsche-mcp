## DATA CATEGORIES

### 1. MODEL SPECIFICATIONS
- Engine type, displacement, bore x stroke
- Power output (hp/kW) at RPM
- Torque (ft-lb/Nm) at RPM
- Weight (curb, dry, GVW)
- Dimensions (L x W x H, wheelbase, track width)
- 0-60 mph / 0-100 km/h
- Quarter mile time and speed
- Top speed
- Fuel economy (city/highway/combined)
- Drag coefficient (Cd)
- Fuel tank capacity
- Transmission type and gear count
- Drive layout (RR, MR, FR, AWD)
- Production years
- MSRP (original and adjusted)
- Production numbers (total and by variant)

### 2. ENGINE DATA (Deep)
- Engine code (M96, M97, MA1, etc.)
- Compression ratio
- Valve configuration (SOHC/DOHC, valves per cylinder)
- Fuel system (carbs, CIS, Motronic, DFI)
- Ignition system
- Firing order
- Idle RPM
- Redline RPM
- Oil capacity (with/without filter change)
- Coolant capacity
- Recommended oil weight and spec
- Timing chain/belt intervals
- Bore and stroke tolerances
- Piston ring gap specs
- Main bearing clearances
- Rod bearing clearances
- Valve clearance specs (shim sizes)
- Head bolt torque sequence
- Rebuild clearances and tolerances

### 3. TRANSMISSION DATA
- Gear ratios (1st through 6th/7th/8th)
- Final drive ratio
- Fluid type and capacity
- Clutch specifications
- Shift cable adjustment specs
- PDK/Tiptronic specific data
- Limited slip differential specs

### 4. TORQUE SPECIFICATIONS
- Engine (head bolts, main caps, rod bolts, cam caps, intake/exhaust manifolds)
- Suspension (control arm bolts, strut mounts, sway bar links, tie rods)
- Brakes (caliper bolts, rotor set screws, master cylinder)
- Wheels (lug nut torque by model)
- Drivetrain (axle nuts, CV joints, flywheel bolts, pressure plate)
- Body (subframe bolts, engine mount bolts, transmission mount)
- Steering (rack bolts, column, pump)

### 5. FLUID SPECIFICATIONS
- Engine oil (type, capacity, filter part number)
- Transmission fluid (manual, PDK, Tiptronic)
- Differential fluid
- Power steering fluid
- Brake fluid (DOT spec)
- Coolant (type, capacity, mix ratio)
- Washer fluid
- AC refrigerant (type, capacity)

### 6. MAINTENANCE SCHEDULES
- Service intervals by model (minor/major)
- What's included at each interval
- Timing chain/belt replacement intervals
- Spark plug replacement intervals
- Brake fluid flush intervals
- Coolant replacement intervals
- Air filter replacement
- Cabin filter replacement
- Drive belt replacement

### 7. COMMON ISSUES & KNOWN PROBLEMS
- IMS bearing failure (996/997)
- Bore scoring (996/997 3.4/3.6)
- RMS leak (996/997)
- D-chunk cylinder liner issues (993)
- Valve guide wear (air-cooled)

- Coolant pipe failure (Cayenne)
- AOS failure (997)
- Headliner sag
- Window regulator failure
- Variocam issues
- PDK mechatronic unit failures
- Turbo wastegate rattle (997.2 Turbo)
- Each issue: affected models, symptoms, cost to fix, DIY difficulty

### 8. PAINT CODES
- Every paint code by year and model
- Color name (German and English)
- Interior leather color codes
- Special/PTS (Paint to Sample) colors
- Color popularity by year

### 9. OPTION CODES
- Every Porsche option code (M-codes, X-codes)
- Description
- Applicable models and years
- Current desirability rating for resale

### 10. VIN DECODER
