Setup in 60 Seconds

Works with Claude Desktop, Hermes Agent, or any MCP-compatible client.

# Install
pip install git+https://github.com/softwealth/porsche-mcp.git

# Claude Desktop — add to config.json:
{
  "mcpServers": {
    "porsche": {
      "command": "reno-porsche-mcp"
    }
  }
}

# Hermes Agent — add to config.yaml:
mcp_servers:
  porsche:
    command: "reno-porsche-mcp"

Then just ask your AI: "What are the torque specs for a 997 GT3?"
